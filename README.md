# PSO-SVR-GMPEs
A near-fault ground-motion prediction equations for constant-strength relative input energy based on machine learning

A support vector regression (SVR) framework is employed to predict constant-strength relative input energy, with model hyperparameters globally optimized using the particle swarm optimization (PSO) algorithm. 
PSO-SVR GMPEs.rar contains flat file for the near-fault ground motion database, training models and calculation examples.
